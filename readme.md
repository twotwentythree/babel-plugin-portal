# babel-plugin-portal [![Build Status](https://travis-ci.com/twotwentythree/babel-plugin-portal.svg?branch=master)](https://travis-ci.com/twotwentythree/babel-plugin-portal)


## Install

```
$ npm install babel-plugin-portal
```


## Usage

```json
{
	"plugins": [
		[
			"portal",
			{
				"portals": [
					"create-fetcher.js"
				]
			}
		]
	]
}

```
